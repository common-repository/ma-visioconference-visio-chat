Service de visioconférence permettant le partage de caméras, micros, écran et documents.

== Description ==

Avec le module Ma-Visioconférence pour Wordpress, ajoutez simplement votre salle de réunion virtuelle directement sur votre site, sur n'importe quel contenu Wordpress.

Planifiez votre réunion, Visio Gratuite à 3 participants, ou Visio Pro pour réunir plus d'invités, et insérez un lien ou l'encart de visioconférence à vos posts.

Comment ça marche ?

1) L'interface vous permet de planifier directement les réunions virtuelles depuis votre administration Wordpress, lister vos réunions, recharger votre compte...

2) Vous n'avez qu'à ajouter le lien vers votre réunion via le bouton dédié sur la page de rédaction de contenu

Les fonctionnalités ?

Une salle de visioconférence à inclure directement sur votre site, partage de webcam, micro, écran et documents.

Pour plus d'informations sur le service, et pour créer votre compte en ligne (nécessaire), visitez www.ma-visioconference.fr

== Installation ==

1. Installez le plugin
2. Activatez le plugin depuis la section 'Plugins' de votre console d'administration Wordpress
3. Créez un compte sur www.ma-visioconference.fr pour obtenir votre clé API (via la section 'Profil')
4. Insérez votre clé API depuis le menu 'Ma-Visioconférence' de votre console d'administration Wordpress
5. Planifiez une visioconférence depuis le menu 'Ma-Visioconférence' de votre site Wordpress
6. Insérez le lien public (ou la visioconférence) grâce aux boutons présents dans l'interface de création de post (bouton sur l'encart Wysiwyg)

== Frequently Asked Questions ==

= Doit on créer un compte sur www.ma-visioconference.fr ? =

Oui, afin d'obtenir votre clé API.

= Le service est il gratuit ? =

Oui, deux types de réunions existent : la Visio Gratuite et la Visio Pro. Le service gratuit vous permet de créer des visioconférences à 3 participants et de partager audio, vidéo, écrans et documents.

== Screenshots ==
